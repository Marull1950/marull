Odoo Website Display Packaging Options

Display Packaging Options on Website Product Page!!

Odoo Website Display Packaging Options Module enables you to publish multiple packaging options for a product on Odoo Website. It allows customers to select the packaging type directly from the product page. Each packaging contains a fixed number of a particular product.

Combine Similar Packagings In A Website Order For delivery
Automatically combine similar packaging order lines in the website order during delivery generation.
If a manufacturing Order is created for the website order, the packaging is combined together into a single orderline in MO.
Merge multiple MOs With different Packaging of same product in Odoo
Manually merge multiple separate Manufacturing orders into one, if they have packaging of the same product.
The merged Odoo MO also records the details of each of the original manufacturing orders.
The corresponding BOM of the Product is automatically merged to show required Quantities to make merged packages.
