Overview

A Smoother Shopping Experience In Odoo's Shopping list application, users can create a shopping list of their own choice from website portal itself. For example, you are ordering some regular grocery items like bakery, oil, cereals, dairy products, food packages, nuts, seeds, and fruits. You will have to add all these items every time separately. But, upon installing the custom website module, you can create a shopping list wherein you can add all these items with its quantity and directly add the list to cart to remove all the hassles.

Features

Create your Regular or frequently bought product's Shopping lists once and place order as many times as you wish.

Create, manage, and modify shopping lists any time

Easy user interface

Screenshots

Users can create a shopping list by clicking on the "create shopping list" button.

Upon initiating shopping list creation, a shopping list popup would be triggered asking you to enter the name of particular shopping list. For example: Kitchen List, Furniture List, Medical List, etc.

Once you have created different shopping lists you can start adding products according to your requirements. You will notice a "Add to Shopping List" button besides the product.

Upon hitting the "add to shopping list" button a pop-up will be displayed asking you to select a shopping list from the drop down that you would like to add that product to along with its quantity.

You can view the shopping list before moving ahead.

If user tries to add product that is already present in particular shopping list an error will occur.

User can add/remove product from the shopping list.

User can hit the cart button besides the shopping list to add the whole shopping list to the cart.

User can hit the remove button besides shopping list to discard particular shopping list.

Upon hitting remove shopping list button you will see a pop-up asking you about the confirmation of deleting the shopping list.

User can also view the shopping list menu from backend.

User is able to view the shopping list tree view.

User can also view the shopping list form.

User can also add, edit, or delete shopping list from their mobile devices.
