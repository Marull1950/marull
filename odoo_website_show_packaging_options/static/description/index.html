Display Packaging Options on Website Product Page


Create Multiple Product packaging options in Odoo for a product. Publish these options on Odoo Website.

Allow customers can select the packaging type directly from the product page.

Each packaging contains a fixed number of a particular product.

The module works in conjunction with Odoo Merge Similar Packaging Orders module to merge similar packaging orderline in an order.

Combine Similar Packagings In A Website Order For delivery.

Automatically combine similar packaging orderlines in the website order during delivery generation.

If a manufacturing Order is created for the website order, the packaging are combined together into single orderline in MO.


Merge multiple MOs With different Packaging of same product in Odoo

Manually merge multiple separate Manufacturing orders into one, if they have packagings of same product.

The merged Odoo MO also records the details of each of the original manufacturing orders.

The corresponding BOM of the Product is automatically merged to show required Quantities to make merged packagings.
